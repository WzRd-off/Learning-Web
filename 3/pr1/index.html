Фразеологізми

Бити байдики — ледарювати; зарубати на носі — запам'ятати; пекти раків — червоніти від сорому; ні пари з уст — мовчати.

Типи складних речень

ССР: частини рівноправні (і, а, але). СПР: є головна і залежна частина (що, бо, який). БСР: частини поєднані інтонацією, без сполучників.

Математика

Формули скороченого множення

(a±b)² = a²±2ab+b².
Різниця квадратів: a²-b² = (a-b)(a+b).
Сума/різниця кубів: a³±b³ = (a±b)(a²±ab+b²).

Властивості логарифмів

logₐ(xy) = logₐx + logₐy.
logₐ(x/y) = logₐx - logₐy.
logₐ(xⁿ) = n·logₐx.

Основи тригонометрії

Основна тотожність: sin²α + cos²α = 1.
У прямокутному трикутнику: sinα = протилежний катет / гіпотенуза; cosα = прилеглий катет / гіпотенуза.
